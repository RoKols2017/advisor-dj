# Development Plan

## Milestones

1. Стабильность импорта: watcher + ручной импорт + карантин.
2. Качество: unit/integration тесты и удержание CI quality gates.
3. Production hardening: proxy/TLS, deploy checklist, observability.

- Q1: завершить стабилизацию импортов и унификацию ошибок.
- Q2: расширить интеграционные тесты и покрытие watcher-сценариев.
- Q3: усилить прод-эксплуатацию (бэкапы, алерты, метрики).
